# StreamNotify

- YouTube/ニコニコ生放送/Twitchの放送開始を自動検知し、  
Bluesky へリアルタイムにお知らせ投稿する Python 製 Bot です。
- ニコニコは動画投稿、YouTubeは放送と動画投稿の通知に対応します。
- **v3系**をメインバージョンとして開発されており、GUIからの操作や、  
プラグインによる機能拡張に対応しています。

---

## 主な特徴 (v3)
### 基本機能
- **YouTube 監視 (RSS/WebSub)**: 指定チャンネルの新着動画を自動検出（リアルタイムプッシュ通知対応）
- **ニコニコ動画/YouTubeLive**の検知に対応（プラグイン経由）
- **統合設定画面**: GUI上で全ての設定を一元管理可能
- **バックアップ・復元**: 設定やデータベースをGUIから簡単にバックアップ可能

### 投稿関連
- **Bluesky**へ自動で放送開始/終了通知を投稿
- **Jinja2 テンプレート**による柔軟な投稿内容のカスタマイズ
- **画像添付・リンクカード**対応（Bluesky API連携）
- **一括投稿予約**: 複数の動画をスケジュールに基づいて予約投稿可能

### 安全性・保守機能
- **堅牢な設計**: 拡張性を考慮したプラグインアーキテクチャと、SQLiteによる安定したデータ管理

---

## バニラ (コア) と プラグイン
本アプリは「最小限の構成（バニラ）」から、用途に合わせて拡張可能です。

- **バニラ状態**: YouTube RSS監視とBlueskyへのシンプル投稿（基本機能のみ※1,2）
- **プラグイン導入後**: 画像リサイズ投稿、ニコニコ連携、YouTube詳細API連携、  
高度なロギングなどが有効化されます。
> ※1 サイト連携機能については、初回起動時は無効化されています。  
> GUIから適切な設定(アカウント設定)を行うことで有効化できます。  
> ※2 画像投稿機能とログ拡張機能は、デフォルトで有効になっています。  
必要に応じて設定の調整を行ってください。

## 起動方法 (v3)
1. Releaseページから最新の安定版をダウンロードし、ZIPアーカイブを解凍します。
2. 依存パッケージをインストールします。
   ```bash
   pip install -r v3/requirements.txt
   ```
3. メインスクリプトを起動します。
   ```bash
   python v3/main_v3.py
   ```
- ※次期バージョンでは、より簡単にセットアップ・起動を行っていただけるように、    
セットアップ・修復・起動ランチャーを導入予定です。

## 運用上の注意とセキュリティ
- **マニュアルや技術ドキュメントは添付されていません**:  
ユーザーガイド、GUI操作マニュアル、トラブルシューティング、技術ドキュメントなどは、  
直接Releaseには含んでいません。これらの内容を確認するには[Wiki](https://git.neco-server.net/mayuneco/StreamNotify/wiki)をご確認ください。
- **APIキーの秘匿**:   
セキュリティのため、**APIキーやパスワード**は絶対に公開リポジトリに含めないでください。  
また、それらの内容は **設定ファイル(`settings.env`)** や、**バックアップファイル（`backup/`以下）**、  
**設定ファイルのバックアップ（`settings.env.backup`）** に含まれています。
- **リソース監査**:  
ログファイル（`logs/`以下）には重要な履歴が記録されます。  
運用時はアクセス権限に注意してください。

本アプリは個人運用・検証を想定しています。商用利用は自己責任でお願いします。
