# 画面操作マニュアル (GUI の使い方)

このページでは、StreamNotify の専用画面（GUI）の基本的な使い方について説明します。  
直感的な操作で、動画の確認から投稿、細かな設定までをすべて画面上で行うことができます。
> この文書は現在の最新バージョンであるv3.2.0を基準としています。  
> そのため既存バージョンを利用中の場合、一部の機能や説明が異なる場合があります。

---

## 🖥 画面の基本構成

StreamNotify を起動すると、大きく分けて 3 つのエリアが表示されます。

1. **ツールバー (上部)**  
   更新、投稿テスト、各種設定の呼び出しなど、アプリの主な操作を行うボタンが並んでいます。
2. **フィルタパネル (中央上)**  
   「未投稿の動画だけ見たい」「YouTubeの配信だけ見たい」といった条件で、  
   動画リストを絞り込むためのエリアです。
3. **動画リスト (中央〜下部)**  
   取得した動画の一覧が表示されます。ここで投稿したい動画を選んだり、詳細を確認したりします。

---

## 🖱️ ツールバーの主要機能

上部に並んでいるボタンを使って、日常の操作を行います。

| ボタン | 機能と役割 |
| :--- | :--- |
| **🔄 再読込** | データベースから最新のリストを読み込み直します。 |
| **📡 新着取得** | YouTube やニコニコ動画をチェックし、最新情報を取得します。<br>（自動取得を待てない場合に使用します）。 |
| **🎬 Live判定** | YouTube のライブ配信（直近24時間以内）の状態をチェックし、<br>放送中かアーカイブ済みかを最新の状態に更新します。 |
| **🧪 投稿テスト** | 選択した動画を「テスト投稿」します。<br>Blueskyには実際には投稿されず、どのような文章になるかをログで確認できます。 |
| **📤 投稿実行** | 選択した動画を Bluesky に実際に投稿します。<br>（※自動投稿設定時は使えません） |
| **📅 一括予約** | 複数の動画を選び、「〇時から〇分間隔で順番に投稿する」といった予約を行います。 |
| **⚙️ 全体設定** | アプリの設定 (`settings.env`) を画面から直接編集します。 |
| **📝 テンプレート** | 投稿文面（テンプレート）を編集・プレビューできる画面を開きます。 |
| **📊 統計** / **💾 バックアップ** | 過去の投稿数を確認したり、データのバックアップを取ります。 |

> [!TIP]
> 動作モードが `autopost` (自動投稿) の場合、  
手動での「📤 投稿実行」などのボタンはグレーアウトして押せなくなります。

---

## 🔍 見たい動画を探す（フィルタ機能）

動画が増えてきたら、フィルタを使って目的の動画をすぐに見つけられます。  
設定を変更すると、リストが瞬時に切り替わります。

- **テキスト検索**: タイトルの一部などを入力して検索します。
- **投稿状態**: 「すべて」「未投稿」「投稿済み」から絞り込みます。
- **配信元**: 「すべて」「YouTube」「Niconico」から選びます。
- **種別**: 「動画」「ライブ中」「予約枠」「アーカイブ」など、種類ごとに表示を切り替えます。

---

## 📋 動画リストと右クリック操作

一覧には動画のタイトル、公開日時、種類（アイコン）、投稿状況などが表示されます。  
動画を選んで **右クリック** すると、さらに便利なメニューが開きます。

- **⏰ 予約日時を設定**:  
投稿したい日時をカレンダーと時計から個別に指定できます。
- **🖼️ 画像を設定**:  
パソコン内の画像を選んでサムネイルにしたり、URLから画像を取得して設定し直すことができます。
- **🗑️ 削除**:  
リストから動画を消去します。※間違えて取得してしまった動画や、もう表示させたくない場合に便利です。
- **❌ 選択解除**:  
設定した予約時間やカスタム画像をリセットします。

---

## 💡 便利な使い方：YouTube Live 手動判定

ツールバーにある **「🎬 Live判定」** ボタンは、特にライブを見逃したくない時に役立ちます。

通常、アプリは一定時間ごとにチェックを行いますが  
「予約枠から放送中に変わったか、今すぐ確認したい！」という場合には、  
このボタンを押すことで最新の情報を強制的にチェックし、正確な状態に素早く更新できます。
