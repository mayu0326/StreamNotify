# 📖 用語集 (Glossary)

StreamNotify の設定や操作で出てくる、ちょっと難しい用語の解説です。

---

## 基本用語

- **v3 統合版 (StreamNotify v3)**  
  v3.2.0のアプリのことです。  
  v3.3.0-alphaやv3.3.0-betaと呼ばれていた実験中機能などを統合したため、統合版と呼ばれています。  
  これらの実験中・検証中機能が一つにまとまり、設定も画面（GUI）から簡単にできるようになりました。

- **GUI (Graphical User Interface)**  
  マウスでボタンを押したり画面を見ながら操作できる、いわゆるアプリ画面のことです。  
  「黒い画面（コマンドプロンプトやターミナル）」での操作の対義語です。

- **settings.env**  
  アプリの動作設定（パスワードやIDなど）が保存されているファイルの名前です。  
  このファイルには**すべての設定やアカウント情報**が保存されています。  
  バージョンアップや再インストールの際は、  
  このファイルをコピーしておけば設定を引き継ぐことができます。

- **データベース (video_list.db)**  
  これまでに見つけてきた動画のリストや、  
  「もう Bluesky に投稿したかどうか」の記録（投稿済みフラグ）が保存されている  
  倉庫のようなファイルです。`data`フォルダの中にあります。

## 投稿・動作に関する用語

- **手動モード (`selfpost`) / 自動モード (`autopost`)**  
  アプリの動作モードです。「自分で選んで投稿ボタンを押すか」「全自動で任せるか」の違いです。  
  詳しくは [自動投稿（Autopost）機能の解説](autopost_guide.md) をご覧ください。

- **投稿テスト / ドライラン (Dry-run)**  
  「実際には SNS へ投稿しないで、どういう文章や画像になるか（リサイズがうまくいくかなど）を試す」  
  ためのお試し機能です。

- **Jinja2 (ジンジャツー)**
  投稿の文章を作るための「テンプレート」の仕組み（エンジン）の名前です。  
  `{{ title }}` のような文字を、実際の動画タイトルなどに置き換えてくれます。

## YouTube・動画取得に関する用語

- **RSS（アールエスエス）**  
  YouTube やニコニコ動画が用意してくれている「新着動画のお知らせ一覧表」のようなものです。  
  StreamNotify はこれを定期的に見に行く（**ポーリング**する）ことで動画を見つけています。

- **ポーリング**  
  一定間隔で「今どうなってる？」と確認しに行くことです。  
  初期設定では15分毎に確認することになっています。

- **WebSub（ウェブサブ）**  
  RSS よりも高速に「新着動画が出た瞬間に YouTube 側からプッシュ通知を送ってくれる」仕組みです。  
  現在は一部のサポーター限定のベータ機能として提供している機能です。（[詳細はこちら](websub_guide)）

- **チャンネル ID**  
  YouTube における「会員番号」のようなもので、必ず `UC` から始まる 24 文字の英数字です。  
  チャンネル名（例：`まゆチャンネル`）やチャンネル URL ではありません。  
  また、ハンドル(`@mayu`)を使うにはAPIキーが必要です。  これを間違えると動画が一切取得できません。
  
- **YouTubeAPI/APIキー**  
  - YouTubeのサーバー受付のようなものです。正式名称は「YouTubeDataAPI(v3)」です。  
  - APIキーは、YouTubeのサービスを利用するために必要なパスワードのようなものです。  
  - チャンネルIDを使う場合は不要ですが、ハンドルを使う場合は必要です。
  - 動画やLiveの詳細情報(タイトルやサムネイルなど)を取得するにはAPIキーが必要です。

  詳しくは [YouTubeAPIキーの取得方法](youtube_api) をご覧ください。

- **ライブ判定 / Live Detection**  
  YouTube の動画が「アップロード済み」なのか、「これから配信（待機所）」なのか、「現在配信中」なのかを、  
  API を使って細かく見分ける機能です。

- **プレミア公開**
  YouTube で、録画した動画をテレビ番組のように「決まった日時にみんなで一斉に見る」機能です。  
  最初だけはライブ配信のようにリアルタイムで、その後は普通のアーカイブ動画になります。
