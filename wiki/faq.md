# よくある質問 (FAQ) とトラブルシューティング

StreamNotify を使っていて困った時の解決策や、よくある質問をまとめました。

---

## 🔴 アプリが起動しない・エラーが出る

### Q: 「ModuleNotFoundError」というエラーが出る
**A**: アプリを動かすための「専用の部品（ライブラリ）」が不足しています。  
コマンドプロンプトやターミナルを開き、アプリのフォルダ（仮想環境内）で、  
以下のコマンドを実行してインストールしてください。
```bash
pip install -r requirements.txt
```

### Q: メモリの使用量が増え続けて、アプリが重くなる
**A**: 何千件もの動画が蓄積しているか、ログファイルが巨大化している可能性があります。  
設定から「キャッシュをクリア」するか、何日かに1回、定期的にアプリを再起動することをおすすめします。

## 🟠 動画が取得できない

### Q: YouTube の動画が一覧に全く表示されない！
**A**: 以下の点を確認してみてください。
- **チャンネル ID の間違い**:  
`settings.env`（または設定画面）のチャンネル ID が `UC` から始まる 24 文字の英数字に  
なっているか確認してください。  
(`@` から始まるハンドル名は、YouTube API キーを設定している場合のみ使えます)
- **インターネット接続**:  
セキュリティソフトなどがアプリの通信をブロックしていないか確認してください。

### Q: ニコニコ動画が取得できない
**A**: ツールバーの「🔧 プラグイン」ボタンから、  
ニコニコプラグインが有効（オン）になっているか確認してください。  
また、ニコニコのユーザー ID（数字）が正しく設定されているか見直しましょう。

## 🔵 Bluesky への連携・投稿について

### Q: Bluesky に投稿できない（認証失敗と出る）
**A**: 設定した内容に間違いがある可能性が高いです。以下をチェックしてください。
1. **ハンドル名**: 先頭の `@` を抜いた形式（例: `yourname.bsky.social`）になっていますか？
2. **パスワード**: 普段のログインパスワードではなく、必ず「**アプリパスワード**」を使っていますか？
3. **コピペのミス**: 文字の最後や途中に、余計な空白スペースが入っていませんか？

解決しない場合は、Bluesky 側で一度アプリパスワードを作り直して設定し直してみてください。

### Q: 投稿はされるけど、画像（サムネイル）が出ない
**A**: アプリのフォルダの中に `images/default/noimage.png` という画像ファイルが存在するか確認してください。  
また、極端に大きなサイズの画像は Bluesky 側の制限（1MB）で弾かれることがあります。

## 🟢 画面の操作・その他

### Q: 検索（フィルタ）をしたら何も表示されなくなった
**A**: 複数のフィルタ機能が重なって掛かっている可能性があります。  
フィルタパネルの **「🔄 リセット」** ボタンを押して、すべての検索条件を一度クリアしてみてください。

### Q: 同じ動画が2回投稿されてしまう
**A**: 基本設定で「過去の投稿と重複させない（`PREVENT_DUPLICATE_POSTS`）」が オフになっている可能性があります。  
また、テストモード（dry_run）中は、テストを何度でも行えるように重複チェックの機能が無効になります。  
手動投稿モード（selfpost）に戻っているか確認してください。
