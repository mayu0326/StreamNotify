# 🔑 YouTube チャンネル ID の調べ方

StreamNotify で特定の YouTube チャンネルの新着動画やライブ配信を監視するには、そのチャンネル固有の「**チャンネル ID**」が必要です。

> [!WARNING]
> `@` から始まる「ハンドル名（例：@yourchannel）」や、URL の末尾にあるカスタム文字列ではありません。  
> 必ず `UC` から始まる 24文字の ID を取得してください。

---

## 🔍 チャンネル ID の取得手順

一番確実で簡単な方法は、ウェブブラウザで YouTube を開いて自分のアカウント設定から確認する方法です。

### 手順
1. パソコンのブラウザで [YouTube](https://www.youtube.com/) にアクセスし、ログインします。
2. 画面の右上にある **「プロフィールアイコン（自分のアイコン）」** をクリックします。
3. メニューの中から **「⚙️ 設定」** をクリックします。
4. 左側のメニューの一番下にある **「詳細設定」** をクリックします。
5. 画面の中央に **「チャンネル ID」** という項目が表示されます。
   （※ `UC` から始まる 24文字の英数字の列です）
6. その横にある **「コピー」** ボタンを押してコピーします。

これでチャンネル ID の取得は完了です！

---

## 📝 アプリへの設定方法

取得したチャンネル ID は、StreamNotify の画面（GUI）から設定するか、設定ファイルに直接書き込みます。

### 画面（GUI）から設定する場合（おすすめ）
1. アプリを起動し、ツールバーの **「⚙️ 全体設定」** をクリックします。
2. **アカウント** > **YouTube** タブを開き、「チャンネル ID」の欄に先ほどコピーした ID を貼り付けて保存します。

### 設定ファイル（`settings.env`）に直接書く場合
`settings.env` をメモ帳などで開き、以下のように `YOUTUBE_CHANNEL_ID=` の右側に貼り付けます。

```env
# 監視対象のチャンネル ID（UC で始まる 24 文字）
YOUTUBE_CHANNEL_ID=UCxxxxxxxxxxxxxxxxxxxxxx
```
