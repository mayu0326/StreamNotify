# 🔑 YouTube API キーの取得方法

StreamNotify では API キーを使わなくても基本的な機能（新着動画のチェックなど）は問題なく動きます。  
しかし、**YouTube API キー** を設定することで、以下のような**より高度な機能**が使えるようになります！

- `@` から始まるハンドル名（例 `@yourchannel`）でのチャンネル指定
- YouTube動画、Liveの詳細情報の取得(タイトル、サムネイルなど)
- YouTube ライブの「放送前（予約）」「放送中」「放送終了」の正確な自動判定と追跡

> [!NOTE]
> API キーの取得には Google アカウントが必要です。設定には少し手順がかかりますが、無料で利用できます。

---

## 🔍 API キーの取得手順

Google の開発者向けページ（Google Cloud Console）にアクセスしてキーを発行します。

### 手順 1. プロジェクトの作成
1. パソコンのブラウザで [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Google アカウントでログインします。
2. 画面左上にある「プロジェクトの選択（または現在のプロジェクト名）」のプルダウンをクリックし、**「新しいプロジェクト」** を選びます。
3. プロジェクト名に分かりやすい名前（例：`StreamNotify`）を入力し、**「作成」** をクリックします。
4. 少し待つとプロジェクトが作成されるので、再度左上のプルダウンから作成したプロジェクトを選択します。

### 手順 2. YouTube Data API の有効化
1. 左側のハンバーガーメニュー（三本線）を開き、**「API とサービス」** > **「ライブラリ」** をクリックします。
2. 検索窓で **`YouTube Data API v3`** と検索し、出てきた結果をクリックします。
3. 青い **「有効にする」** ボタンをクリックします。

### 手順 3. API キーの発行
1. 「YouTube Data API v3」の管理画面に移動したら、画面左側のメニューから **「認証情報」** をクリックします。
2. 画面上部の **「＋ 認証情報を作成」** をクリックし、**「API キー」** を選びます。
3. 画面に「API キーを作成しました」と表示され、`AIzaSy...` から始まる長い文字列が表示されます。これがあなたの **API キー** です。
4. 横にあるコピーボタンを押して、キーをコピーしてください。

これで API キーの取得は完了です！

---

## 📝 アプリへの設定方法

取得した API キーを StreamNotify に設定します。

### 画面（GUI）から設定する場合（おすすめ）
1. アプリを起動し、ツールバーの **「⚙️ 全体設定」** をクリックします。
2. **アカウント** > **YouTube** タブを開き、「YouTube API キー」の欄にコピーしたキー（`AIzaSy...`）を貼り付けて保存します。

### 設定ファイル（`settings.env`）に直接書く場合
`settings.env` をメモ帳などで開き、以下のように入力します。

```env
# YouTube API キー（高度なライブ判定などに使用）
YOUTUBE_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

設定後、アプリを再起動すると、YouTube ライブの細かい状態変更（待機所作成 → 放送開始 → アーカイブ化）などが正確に検知・追跡されるようになります！
