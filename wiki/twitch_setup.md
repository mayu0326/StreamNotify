# Twitch 連携設定ガイド（StreamNotify v4）

このページでは、**Twitch（ツイッチ）** の配信開始・終了・Raid などの情報を StreamNotify **v4** で受け取り、<br>Bluesky に投稿するまでの流れを説明します。

> [!IMPORTANT]
> - **Twitch 連携は v4 専用**です。<br>v3 では**Twitch連携ができません**のでバージョンアップをお願いします。  
> - **「定期チェックだけ」モード（`poll`）では、YouTubeとニコニコ動画の通知だけ使えます。**<br>
> - **Twitch の通知機能**を使うには、サーバー(センター)と **通知が早く届く連携（WebSub）** を有効にする必要があり、<br>センターと通信できる状態でのみ利用できます。

---

## 🌟 何ができるか

連携が成功すると、次のようなタイミングで情報を受け取り、テンプレートにしたがって Bluesky へ投稿できます<br>（動作モードが `autopost` などの設定次第）。

- 配信が始まったとき（配信開始）
- 配信が終わったとき（配信終了）
- Raid など、サーバーが送るイベントに対応している場合

投稿文のひな型は **`v4/templates/twitch/`** にあります。<br>編集方法は [テンプレートのカスタマイズ](template) ・ [テンプレートの変数リスト](template_arguments) を参照してください。

---

## ✅ 事前にそろえるもの

1. **StreamNotify v4 が普通に動く状態**  
   [v4 クイックスタート](v4_getting_started) まで完了していること。
2. **`YOUTUBE_FEED_MODE=websub`**  
   設定ファイル（`settings.env`）または **「⚙️ 設定」** 画面の基本設定で、YouTube の取得方法が **websub** になっていること。  
   **`poll` のときは Twitch 連携・EventSub は動きません。**
3. **通知サーバー（センター）利用のための ID とキー**  
   [WebSub（リアルタイム通知）の利用手順](websub_guide) のとおり、サーバー設定が正しく行われていること。
4. **PC が通知を受け取れること**  
   アプリは裏で **ローカルの待ち受け（Webhook）** を動かし、センターから Twitch 関連の通知を受け取ります。  
   `WEBSUB_CALLBACK_BASE_URL`（例: `http://localhost:20000`）が、実際の待ち受けポートと合っているか確認してください。<br>セキュリティソフトやファイアウォールでブロックされていないかも、うまくいかないときは確認ポイントです。

> [!TIP]
> センターに接続できず **RSS フォールバック(機能制限モード)** に入っているあいだは、**Twitch 連携は無効** になります。<br>メイン画面に **「🔌 WebSubに再接続」** が出たら、回線や設定を直したうえで復帰を試してください（ [v4 GUI 操作マニュアル](v4_gui_usermanual) ）。

## 🛠️ 連携手順（画面から）

1. v4 を起動し、ツールバーの **「⚙️ 設定」** を開きます。
2. **アカウント** のタブから **「Twitch」** を選びます。
3. **「Twitch と連携する」**（または Twitch 連携のボタン）を押します。
4. ブラウザが開き、**Twitch 側のログイン・許可** を求められたら、画面の指示に従います。  
   認証は **センター経由の OAuth** で行われます。  
   `settings.env.example` に書いてある **TWITCH_CLIENT_ID は置いてあるだけで機能しません。**
5. 連携が完了すると、設定画面に **連携状態** が表示されます。  
   連携後はアプリを起動していないときでもサーバー側で、配信開始・タイトルやカテゴリ変更・配信終了・Raid先を取得します。  
   (サーバー側の機能として情報を自動的にBlueskyに投稿する機能はありません。)  

連携をやめたいときは、同じ Twitch タブの **「連携解除 (Disconnect)」** から解除できます。

---

## 📝 テンプレートについて

Twitch 用のデフォルトひな型は次のようなファイルです（フォルダは `v4/templates/twitch/`）。

- 配信開始通知向け
- 配信終了終了向け
- Raid で終了した場合向け

> [!TIPS]
> - Raidを受け取ったときのBluesky投稿機能と、タイトル変更やカテゴリ変更時のBluesky投稿機能はありません。  
> - 詳しい変数名は [テンプレートの変数リスト](template_arguments) を参照してください。

---

## 🆘 困ったときは

- **Twitch タブがグレーアウトする・連携ボタンが使えない**  
  `poll` モードになっていないか、**WebSub フォールバック中** になっていないか確認してください。
- **連携したのに一覧に出てこない**  
  **「🔄 再読込」** を押す、しばらく待つ、`v4/logs/app.log` にエラーが出ていないか確認してください。
- **センター・Twitch 側のエラー**  
  API キーやクライアント ID の誤り、センターのメンテナンスなどが原因のことがあります。<br> [WebSub 利用手順](websub_guide) とログをあわせて確認してください。
