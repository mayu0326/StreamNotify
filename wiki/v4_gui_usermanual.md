# v4 画面操作マニュアル（GUI の使い方）

StreamNotify **v4** のメイン画面の構成と、ボタン・メニューの意味を説明します。  
全体の流れは [v4 ユーザーマニュアル](v4_user_manual) を参照してください。

> [!NOTE]
> v3 の画面説明は [GUI 詳細マニュアル](gui_usermanual) です。**ボタンの名前が v4 と少し違います**（下表のとおり）。

---

## 🖥 画面の基本構成

大きく分けて次のエリアがあります。

1. **ツールバー（最上部）**  
   再読込・投稿・設定など、よく使う操作のボタンがあります。  
   **YouTube の取り方（poll / websub）** や **通知サーバーとの接続状態** によって、**真ん中付近のボタンが出たり消えたり** します。
2. **フィルタ行（一覧の上）**  
   検索語、サービス、タイプ、投稿状態、並び順などで一覧を絞り込めます。
3. **動画リスト（中央）**  
   取得した動画の一覧です。行をクリックして選択します。
4. **サムネイルプレビュー（一覧の下）**  
   選択中の動画のサムネイル表示エリアです。
5. **フッター（最下部）**  
   読み込み状況などの短いメッセージが表示されます。

---

## 🖱️ ツールバーのボタン一覧（v4）

| ボタン | 役割 |
| :--- | :--- |
| **🔄 再読込** | 一覧を最新の状態に更新します。<br>**通知の早い連携（websub）** を使っているときは、通知サーバー側との同期も含む動きになります。 |
| **🔌 WebSubに再接続** | 使える時にだけ**表示されます** 。(サーバー未接続時やメンテ時など)<br>通知サーバーに一時的につながらず **定期チェック方式へ切り替わっているとき** などに、連携をやり直します。 |
| **📡 新着取得 / RSS更新** | モードや状態に応じて**表示されます** (pollモード時、サーバー障害時、未接続時など)。<br>YouTube などを **手動でチェック** して一覧を更新します。 |
| **🎬 Live判定** | モードや状態に応じて**表示されます** (pollモード時、サーバー障害時、未接続時など)。<br>YouTube ライブ関連の状態を手動で更新します。 |
| **📝 テンプレート** | 投稿文のひな型（テンプレート）を編集・プレビューする画面を開きます。 |
| **📤 投稿** | 選んだ動画の **「📤 投稿設定」** ウィンドウを開きます。<br>ここで **🧪 投稿テスト** や **✅ 確定して投稿** が行えます。 |
| **📅 一括スケジュール** | 複数件をまとめて、時間をずらして予約投稿します。（v3 の **「一括予約」** に相当） |
| **🗑️ 一括削除** | 選んだ行を一覧から削除します。 |
| **🖼️ 画像設定** | **動画を 1 件だけ選んだとき** に使えます。サムネイルまわりの設定を行います。 |
| **📅 投稿予定一覧** | 予約した投稿の一覧を別ウィンドウで開きます。 |
| **⚙️ 設定** | アカウントや動作モード、バックアップなど **詳しい設定** のウィンドウを開きます。 |

> [!TIP]
> 動作モードが **自動投稿（autopost）** のとき、**「📤 投稿」** など手動投稿まわりのボタンが **押せない（無効）** になることがあります。

### v3 との名前の違い（覚え書き）

| v3 での表記（目安） | v4 での表記 |
| :--- | :--- |
| 📤 投稿設定 | 📤 投稿（ウィンドウタイトルは「📤 投稿設定」） |
| 📅 一括予約 | 📅 一括スケジュール |
| ⚙️ 全体設定 | ⚙️ 設定 |

## 🔍 フィルタ行の使い方

一覧のすぐ上にある入力欄とプルダウンで、表示する行を絞り込めます。

- **検索（🔍 の横）**: タイトルなどに含まれる文字で絞り込みます。
- **サービス**: すべて / youtube / niconico / twitch から選びます。
- **タイプ**: 動画・プレミア・配信中・アーカイブ・予約配信などから選びます。
- **投稿状態**: すべて・投稿済み・未投稿。
- **並び順**: 公開日時（新しい順・古い順）、タイトル、チャンネル名など。

右側の **「すべて選択」「解除」「フィルタリセット」** で、選択状態やフィルタ条件をまとめて戻せます。

---

## 📋 一覧の操作と右クリックメニュー

行を **右クリック** すると、次のようなメニューが開きます（表示はバージョンで多少変わる場合があります）。

- **📋 動画詳細を表示**
- **📤 この動画を投稿**
- **🗓️ スケジュール設定**（予約日時の設定）
- **🖼️ 画像を設定**
- **🔗 ブラウザで開く**
- **❌ 削除**

**行のチェック列（☑）** をクリックすると、投稿や一括操作の対象に含めるかどうかを切り替えられます。

---

## 💡 通知が早い連携（websub）と画面の関係

- 正常にサーバーとつながっているときは、<br>**「📡 新着取得 / RSS更新」** や **「🎬 Live判定」** は **ツールバーに出ない(非表示になる)** ことがあります。<br>異常時(サーバ未接続時・障害時・メンテ等で停止時)や **poll** モードでは **ツールバーに出る(表示されます)** ことがあります。
- 通知サーバーに届かず **自動的に定期チェック(poll)へ切り替わった** ときは、**「🔌 WebSubに再接続」** がツールバーに表示されます。

くわしい仕組みは [v4 概要](v4_overview) や [WebSub 利用手順](websub_guide) を参照してください。
