# テンプレートのカスタマイズ (投稿文面の変更)

StreamNotify では、Bluesky に投稿されるお知らせの「文章（文面）」を自分好みに自由に変更することができます。  
これを **テンプレート機能** と呼びます。

このページでは、テンプレートの編集方法や、便利な自動挿入ワード（変数）の使い方について解説します。

---

## 📝 テンプレートの編集方法 (GUIエディタ/v3.2.0以降)

 StreamNotify v3.2.0 以降では、専用の画面から直感的にテンプレートを作成・編集できます。  
 難しいファイルを直接いじる必要はありません。

1. 画面上のツールバーから **「📝 テンプレート編集」** ボタンをクリックします。
2. 編集するテンプレートの **サービス名** をクリックします。(YouTube, ニコニコ動画など)
3. 編集するテンプレートの **種類** をクリックします。(ライブ配信の開始時、予約時、動画の投稿時など)
4. 編集画面が開きます。
   - **左側**: 文章を入力するエリアです。ここを自由に書き換えます。
   - **右側**: 実際の Bluesky でどのように表示されるか、リアルタイムで**プレビュー**が表示されます。
   - **下部**: クリックするだけで「タイトル」や「URL」などの自動挿入ワード（変数）を  
   簡単に追加できるボタンが並んでいます。
5. 編集が終わったら **「保存 (Save)」** をクリックします。  
これで次回から新しい文章で投稿されます！

> [!TIP]
> ライブ配信の開始時、予約時、動画の投稿時など、**状況ごとに別々のテンプレート**が用意されています。  
用途に合わせてそれぞれ編集しましょう。

---

## 📋 使える自動挿入ワード (変数/v3.2.0以前)

> [!TIP]
> この項目は、v2.x.x～v3.1.xのバージョンで利用しているユーザー向けです。

文章の中に `{{ 波カッコ2つ }}` で囲まれたキーワードを入れておくと、投稿時に実際の動画情報に自動で置き換わります。  
エディタ下部のボタンをクリックすれば、手入力しなくても簡単に追加できます。

### YouTube でよく使うキーワード
| 挿入ワード | どんな文字に変わるか | 実際の表示例 |
|:--|:--|:--|
| `{{ title }}` | 動画や配信のタイトル | 【新着】マイクラ実況 Part1 |
| `{{ video_url }}` | 動画へのリンク (URL) | https://www.youtube.com/watch?v=... |
| `{{ channel_name }}` | チャンネルの名前 | まゆチャンネル |
| `{{ published_at }}` | 動画が公開された日付と時間 | 2025-12-18T10:30:00 |

### ライブ配信（YouTube）で使えるキーワード
| 挿入ワード | どんな文字に変わるか |
|:--|:--|
| `{{ live_status }}` | ライブの状態（upcoming / live / completed） |

### ニコニコ動画でよく使うキーワード
| 挿入ワード | どんな文字に変わるか |
|:--|:--|
| `{{ title }}` | 動画のタイトル |
| `{{ video_url }}` | 動画へのリンク (URL) |
| `{{ channel_name }}` | 投稿者の名前 |

## 🌟 ちょっと便利な高度な使い方

StreamNotify のテンプレートは、**Jinja2**（ジンジャーツー）という仕組みを使っているため、  
少し工夫をするとさらに便利になります。

- この機能については、カスタム機能となっているので、GUIのボタンにはない場合がありますが、  
テンプレートエディタの編集画面や、テンプレートファイルにコピー＆ペーストすれば、使うことができます。

👉 その他のカスタム機能(引数/変数)はこちら: [テンプレートの変数リスト](template_arguments)

### 1. 日付の見栄えをよくする
`published_at`（公開日時）などは、そのまま出すと機械的で少し読みづらい場合があります。  
フィルターを使うとキレイな形式に変換できます。

```jinja2
{{ published_at | datetimeformat('%Y年%m月%d日 %H:%M') }}
# 結果: 2025年12月18日 10:30
```

### 2. 深夜配信のための「24時超え」表記
深夜・早朝の配信を告知するとき、「翌日 3:00」ではなく「今夜 27:00」と書きたいことはありませんか？  
StreamNotify にはこれを自動で計算する便利な機能があります。

予約枠の告知などで `{{ extended_hour }}` を使うと、深夜帯の配信を 24時超えの表記にして自然にお知らせできます。

#### 拡張時刻表示（24時以降30時までの時間表示に対応）
```jinja2
{{ "27:00" | extended_time }}           → "03:00"
{{ "27:00" | extended_time_display }}   → "翌日3時"
```
#### 拡張日時範囲表示（推奨）
```jinja2
{{ format_extended_datetime_range(published_at | datetimeformat('%Y-%m-%d'), 27) }}
# 結果: 2025年12月21日27時(2025年12月22日(月)午前3時)
```

### 3. タイトルによって文章を変える (条件分岐)
タイトルに「Live」という文字が含まれているかどうかで、自動的に文章を切り替えるような  
「もしも〜なら (if)」という書き方も可能です。

```jinja2
{% if "Live" in title %}
🔴 今から配信開始！遊びにきてね！
{% else %}
🎬 新作動画を投稿しました！
{% endif %}

{{ title }}
{{ video_url }}
```

## 🆘 エラーになったら？（安心のバックアップ機能）

もしテンプレートの書き方を間違えてしまっても、**投稿がストップすることはありません**。  
StreamNotify は、エラーを見つけると自動的に「標準の安全なテンプレート」を使って投稿を継続します。

いつでもエディタでプレビューを見ながら、安心してカスタマイズを楽しんでください！
