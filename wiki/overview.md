# StreamNotify 概要 (Overview)

このページでは、StreamNotify が**どのようなツールなのか**、**どんな機能があるのか**について分かりやすく紹介します。  
実際のセットアップや使い方については、[クイックスタート](getting_started) や [ユーザーマニュアル](user_manual) をご覧ください。

---

## 🌟 StreamNotify とは？

StreamNotify は、**YouTube** や **ニコニコ動画** のチャンネルを監視し、  
新しい動画やライブ配信が始まったら **Bluesky** へ自動的にお知らせ（ポスト）するツールです。

あなたがお使いのPC上で動作するため、外部のクラウドサービスなどに依存せず、  
安全かつ確実に通知を届けることができます。

### 📌 おもな機能

1. **新着情報の自動キャッチ**  
   YouTube やニコニコ動画から、新しい動画やライブ配信の情報を自動で取得します。
2. **情報の整理と保存**  
   取得した動画情報は内部に保存され、「すでに投稿したか」「まだ投稿していないか」をしっかり管理します。  
   重複して投稿してしまう心配はありません。
3. **自由なお知らせメッセージ**  
   テンプレートを編集することで、Bluesky に投稿する文章（タイトルやURLなど）や、  
   サムネイル画像をつけるかどうかを自由にカスタマイズできます。
4. **自動・手動の切り替え**  
   完全自動で投稿を任せることも、専用の画面 (GUI) を使って自分で選んだ動画だけを投稿することも可能です。

---

## ⚙️ StreamNotify の特徴

StreamNotify は、使いやすさと安全性を重視した設計となっています。

- **ローカルで完結**:  
データベースや設定ファイルなど、すべてのデータはあなたのPC内にだけ保存されます。
- **高速な通知**:  
YouTube の場合、オプション設定を有効にすることで、  
通常の定期チェックよりも速くリアルタイムに近い形で情報を取得できます。
- **拡張機能（プラグイン）**:  
基本的な機能に加えて、「画像付き投稿」や「ニコニコ動画対応」などの拡張機能を組み合わせて利用できます。

---

## 🔄 お知らせが投稿されるまでの流れ

StreamNotify が動画を見つけてから Bluesky に投稿するまでの大まかな流れは以下の通りです。

1. **見つける**:  
YouTube やニコニコ動画の更新を定期的にチェックし、新しい動画を発見します。
2. **振り分ける**:  
見つけた動画が「ライブ配信中」なのか「予約枠」なのか、または「通常の動画」なのかを自動で判断します。
3. **保存する**:  
動画の情報をツール内に記録します。
4. **投稿する**:  
設定された「動作モード」にしたがって、Bluesky へお知らせを投稿します。

---

## 🎛️ 動作モードについて

あなたの使い方にあわせて、以下の4つの動作モードから選ぶことができます。

| モード名 | どのような動きをするか |
| :--- | :--- |
| **手動投稿 (selfpost)** | ツール画面（GUI）から**自分で選んだ動画だけ**を投稿します。 |
| **自動投稿 (autopost)** | 見つけた新しい動画を**すべて自動的**に投稿します。（※短時間の連続投稿を防ぐ安全機能つき） |
| **テスト (dry_run)** | 実際の投稿は行わず、「どのような投稿がされるか」の**シミュレーション**だけを行います。 |
| **情報収集 (collect)** | 動画の情報を取得して保存するだけで、**一切投稿を行いません**。 |
