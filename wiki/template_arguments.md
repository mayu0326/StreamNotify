# 投稿テンプレートの引数（変数）一覧

このページでは、テンプレート編集時に使用できる **自動挿入ワード（引数・変数）** の一覧をまとめています。  
テンプレート内で `{{ 変数名 }}` と記述することで、投稿時に実際の動画情報などに自動で置き換わります。

> [!TIP]
> お使いバージョンがv3.2.0以降の場合は、アプリ画面の「📝 テンプレート」からエディタを開くと、  
画面下部のボタンをクリックするだけでこれらの変数を簡単に入力できます！

---

## 📺 YouTube で使える変数

動画の状態（新着、ライブ開始など）に合わせて、異なるテンプレートごとに使える変数が決まっています。

### 【新着動画】通知 (`yt_new_video_template.txt`)
公開されたばかりの動画をお知らせする際に使います。

| 変数名 | 置き換わる内容 | 例 |
|:--|:--|:--|
| `{{ title }}` | 動画のタイトル | 【マイクラ】建築配信 Part1 |
| `{{ video_id }}` | YouTube の動画 ID | dQw4w9WgXcQ |
| `{{ video_url }}` | 動画の URL | https://www.youtube.com/watch?... |
| `{{ channel_name }}` | チャンネル名 | まゆチャンネル |
| `{{ published_at }}` | 動画が公開された日時 | 2025-12-18T10:30:00 |

### 【配信枠設置 (予約)】通知 (`yt_schedule_template.txt`)
ライブ配信の待機所ができた時にお知らせする際に使います。

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ title }}` | 配信のタイトル |
| `{{ video_url }}` | 配信の URL |
| `{{ channel_name }}` | チャンネル名 |
| `{{ published_at }}` | 配信の**開始予定日時** |

### 【配信開始】通知 (`yt_online_template.txt`)
ライブ配信が実際にスタートした時にお知らせする際に使います。

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ title }}` | 配信のタイトル |
| `{{ video_url }}` | 配信の URL |
| `{{ channel_name }}` | チャンネル名 |
| `{{ published_at }}` | 配信の**開始日時** |

### 【配信終了 (アーカイブ)】通知 (`yt_archive_template.txt`)
ライブ配信が終了し、アーカイブ化された際にお知らせする際に使います。

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ title }}` | アーカイブのタイトル |
| `{{ video_url }}` | アーカイブの URL |
| `{{ channel_name }}` | チャンネル名 |
| `{{ published_at }}` | 配信が行われた日時 |

## 📺 ニコニコ動画 で使える変数

### 【ニコニコ新着動画】通知 (`nico_new_video_template.txt`)

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ title }}` | 動画のタイトル |
| `{{ video_id }}` | 動画 ID (sm...など) |
| `{{ video_url }}` | 動画の URL |
| `{{ channel_name }}` | 投稿者名 または チャンネル名 |
| `{{ published_at }}` | 動画が公開された日時 |

> [!NOTE]
> **ニコニコの投稿者名 (`channel_name`) について**  
> アプリが自動的に最適な投稿者名を検索して取得します。  
一度取得できた名前はアプリの設定(`settings.env`)に保存され、次回からはより素早く表示されます。

## 🎮 Twitch で使える変数（対応予定）

Twitch(v4以降対応予定) が有効な場合に使用できるようになります。

### 【放送開始】 (`stream.online`)

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ broadcaster_user_name }}` | 配信者の表示名（日本語など） |
| `{{ broadcaster_user_login }}`| 配信者のログインID（英数字） |
| `{{ title }}` | 放送タイトル |
| `{{ category_name }}` | カテゴリ名（ゲーム名や雑談など） |
| `{{ started_at }}` | 放送開始日時 |
| `{{ stream_url }}` | 視聴 URL |

### 【放送終了】 (`stream.offline` / Raid対応)

| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ broadcaster_user_name }}` | 配信者の表示名 |
| `{{ channel_url }}` | チャンネルの URL |
| `{{ ended_at }}` | 放送終了日時 |
| `{{ to_broadcaster_user_name }}`| **Raid先**の配信者名（Raid時のみ）|
| `{{ to_stream_url }}` | **Raid先**の URL（Raid時のみ）|

## ⏰ 全て共通で備わっている変数と機能 (v3.2.0+)

どのテンプレートでも自由に使うことができる、  
システム共通の便利ワードや日時を加工する機能（フィルター・関数）です。

> [!TIP]
> この機能はカスタム機能のため、テンプレートエディタのボタンにはない場合があります。  
> もしボタンがない場合は、書かれている内容を、エディタにコピー＆ペーストすれば使えます。

### 共通変数
| 変数名 | 置き換わる内容 |
|:--|:--|
| `{{ current_date }}` | そのお知らせを投稿した時の「日付」 |
| `{{ current_datetime }}` | そのお知らせを投稿した時の「日時」 |

### 日時を加工する便利な機能（フィルター・関数）
文章の中で時間を分かりやすく表現するために、以下のような特別な機能が用意されています。

| 機能の書き方 | 変換の結果・例 |
|:--|:--|
| `{{ published_at \| datetimeformat('%Y年%m月%d日 %H:%M') }}` | **日付フォーマット**: `2025年12月18日 10:30` |
| `{{ published_at \| weekday }}` | **曜日表示**: `金曜日` など |
| `{{ "27:00" \| extended_time }}` | **24時超え表記の正規化**: `03:00` |
| `{{ "27:00" \| extended_time_display }}`| **24時超えの日本語表示**: `翌日3時` |
| `{{ format_extended_datetime_range(published_at \| datetimeformat('%Y-%m-%d'), 27) }}`<br>※第2引数に「27」などを指定 | **拡張日時範囲表示 (推奨)**: <br>`2025年12月21日27時(2025年12月22日(月)午前3時)` |
| `{{ "" \| random_emoji }}` | **ランダムな絵文字**: `🎉` や `🚀` など毎回変わります |
