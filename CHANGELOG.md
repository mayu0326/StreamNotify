# Change Log

All notable changes to this project will be documented in this file.

## [v3.2.1] - 2026-02-25

### 概要
- StreamNotify v3.2.1 (Maintenance Release)
- v3.2.0 で導入された新機能の安定化とコード品質の向上を目的としたパッチアップデートです。

### 改善点
- **コードの健全化**: Mypy 型チェックへの完全対応および静的解析エラーの解消。
- **時刻精度の向上**: JST 変換処理のバグ修正および表示時刻の正確性の向上。
- **GUI コンポーネントの最適化**: 一括投稿予約および統合設定画面の動作安定性の向上。
- **DB スキーマの整合性**: 5カテゴリ分類（動画/アーカイブ/予約/ライブ/終了）の内部フラグ管理の厳密化。

### バグ修正
- 特定の環境で設定ファイルが正しく同期されない問題を修正。
- 予約リストのソート順が一部正しくなかった問題を修正。

## [v3.2.0] - 2026-01-07


### v3.2.0 アップデート

#### 概要
- StreamNotify v3.2.0 をリリースしました。
- v3.1.0 からの大幅アップデートです。

## 新機能

### WebSub (PubSubHubbub) 対応
YouTube の新着検知を RSS ポーリングから **WebSub（プッシュ通知）** モードへ。
- **メリット**: RSS よりも数十秒〜数分早く新着を検知。
- **設定**: `YOUTUBE_FEED_MODE=websub` で切り替え可能。
- **動的ポーリング**: 配信状況に応じて API チェック間隔を自動調整（API クォータを節約）。

### 高度な4条件複合フィルタリング
動画一覧を自在に絞り込み、運用効率を劇的に向上させます。
- **タイトル検索**: リアルタイムな部分一致検索。
- **投稿状態**: 全て / 投稿済み / 未投稿 の切り替え。
- **配信元**: YouTube / Niconico の絞り込み。
- **コンテンツタイプ**: 🎬 動画 / 📹 アーカイブ / 🔴 放送予約 / 🔴 放送中 / ⏹️ 放送終了 / ✨ プレミア の7種類。

### 統合設定画面
- 全ての設定項目を GUI 上で一元管理できるようになりました。
- `settings.env` の直接編集が不要になり、手動編集による設定エラーを防止します。

### 一括投稿予約
- 複数の動画を選択し、特定の間隔や日時でまとめて投稿予約が可能になりました。
- ツールバーの「一括投稿予約」ボタンから利用できます。

### 投稿予定一覧
- 現在予約されている投稿の一覧を確認・管理できます。
- 予約のキャンセルや日時の変更がスムーズに行えます。

## 改善点
- **統計情報の強化**: 配信元別・日別分析に対応。過去7日間の投稿状況をリアルタイム表示。
- **パフォーマンス最適化**: フィルタリング処理と DB アクセスを高速化。
- **GUI 最適化**: フィルタ結果の件数表示やステータスバーの情報を拡充。

## バグ修正
- Live 関連動画のキャッシュ有効期限チェック機構を修正。
- `scheduledStartTime` の JST 変換精度の向上。
- 特定の条件で日本語検索が正しく機能しない問題を修正。

## ⚠️ 注意事項
- 本バージョンは既存のデータベース（v3.0.x及び3.1.0）と完全な互換性がありますが、  
念のためデータベースのバックアップを推奨します。
- Live 予定時刻が変更された際、DB は更新されますが、Bluesky への再投稿は行われない仕様です。
- WebSub のサポートを開始しましたが、websubからのデータ受け取りはユーザー側でのサーバー構築が必要です。  
また、websub機能はベータ版としての提供となります。次期バージョンで正式サポート予定です。

## [v3.1.0] - 2025-12-18

### v3.1.0 アップデート

## 新機能

### GUI フィルタ・検索機能
- タイトル検索、投稿状態フィルタ、配信元フィルタ、種類フィルタを追加。

- `PREVENT_DUPLICATE_POSTS` 設定を追加。

### バックアップ・復元機能
- ZIP形式でDB・テンプレート・設定を一括エクスポート/インポート可能。
- 主な機能:
  - API キー・パスワードの安全な除外オプション
  - 復元時の自動バックアップ（既存ファイル保護）
  - 画像フォルダのオプション包含

## 改善点
- GUI の操作性を向上。
- フィルタリング処理のパフォーマンスを最適化。

## バグ修正
- 特定の条件でフィルタが正しく適用されない問題を修正しました。

## [v3.0.0] - 2025-12-17

### v3.0.0 アップデート

#### 概要
- StreamNotify v3.0.0 をリリースしました。
- v2.1.0 からのアップデートです。

## 新機能

### マルチテンプレート対応
- YouTube Live/アーカイブ/新着動画、Niconico新着動画の基本テンプレートに対応。
- 主な機能:
  - 配信状態（ライブ中/終了/新着）に応じたテンプレート自動切り替え
  - テンプレート未設定時は標準フォーマットを適用

### 高度なフィルタリング
- 配信状態（ライブ/アーカイブ/動画）による内部自動分類を実装。
- （※キーワード指定等の詳細なフィルタリング設定機能は本バージョンには含まれません）

### YouTube 優先度ベースの重複排除
- アーカイブ > ライブ > 新着動画 の優先順位で重複を自動排除。
- より情報の確度が高いアーカイブ情報を優先して保持する仕様に変更。

### AssetManager による自動テンプレート配置
- プラグイン導入時に必要なテンプレート・画像を自動配置。

## 改善点
- テンプレートシステムの安定性を向上。
- プラグイン管理の効率化。

## バグ修正
- 特定の条件でテンプレートが正しく適用されない問題を修正しました。

## [v2.1.0] - 2025-12-28

### v2.1.0 アップデート

#### 概要
- StreamNotify v2.1.0 をリリースしました。
- v2.0.0 からのアップデートです。

## 新機能

### 対応プラットフォーム追加
#### YouTube Live プラグイン
- YouTube Live の開始・終了を自動検知し、Bluesky に投稿可能。
- 主な機能:
  - ライブ配信の開始・終了通知(beta版)
  - アーカイブ動画の自動投稿(beta版)
  - プレミア公開の開始通知(beta版)
  - テンプレート選択機能（yt_online_template.txt / yt_offline_template.txt）

#### ニコニコ動画プラグイン
- ニコニコ動画の新着動画を自動検知し、Bluesky に投稿可能。
- 主な機能:
  - RSS フィードを利用した新着動画の検知
  - 投稿テンプレートのカスタマイズ

### GUI機能・投稿機能
#### 画像登録・投稿機能
- GUIから画像を登録し、Blueskyに投稿可能。
- この機能は、URLリンクカード機能と併用することはできません。
- 主な機能:
  - サムネイル画像の登録
  - サムネイル画像の削除
  - 投稿サムネイルの取得・登録・プレビュー
  - サムネイル画像付き投稿

#### URLリンクカード機能
- 情報取得時に放送URLを自動生成し、投稿時にリンクカードとして表示可能。
- この機能は、画像登録・投稿機能と併用することはできません。
- 主な機能:
  - 放送URLの自動生成
  - 放送URLの登録
  - 放送URLの削除
  - 放送URL付き投稿(リンクカードつき)

#### 画像自動リサイズ機能
- 投稿画像を自動的にリサイズし、Bluesky の要件に適合。
- 主な機能:
  - 縦横比に基づくリサイズ（16:9、4:5、1:1）
  - JPEG 品質の最適化

#### DRY RUN 機能
- 投稿をシミュレートし、実際に投稿せずに動作確認が可能。
- 主な機能:
  - 投稿内容のプレビュー
  - ログにシミュレーション結果を記録

#### 投稿設定ウィンドウ
- GUI で投稿設定を簡単に変更可能。
- 主な機能:
  - 画像添付の有無を選択
  - リンクカードの表示設定

#### 予約投稿機能
- 予約投稿機能を追加。
- 主な機能:
  - 予約投稿の追加
  - 予約投稿の削除

#### サイト連携投稿機能
- サイト連携投稿機能を追加。
- 主な機能:
  - プラットフォームごとのテンプレート設定
  - 指定テンプレートで投稿
  - 画像添付またはリンクカードの表示設定(排他設定)

## 改善点
- ログ出力を強化し、デバッグ情報を詳細化。
- AssetManager のテンプレート配置処理を最適化。
- SQLite データベースのロック処理を最適化。
- Bluesky API のエラーハンドリングを強化。

## バグ修正
- 特定の条件で画像が正しくリサイズされない問題を修正。
- 投稿設定ウィンドウでのクラッシュを修正。
- 特定の条件で投稿が失敗する問題を修正。
- ニコニコ動画のユーザー名取得ロジックを改善。
- ログ出力の改善。
- 予約投稿機能の改善。

## アップデート方法
1. 最新バージョンをダウンロードしてください。
2. `settings.env` をバックアップしてください。
3. アプリケーションを再起動してください。

## 注意事項
- 本バージョンは既存のデータベース構造に影響を与えません。必要があれば自動でマイグレーションされます。
- 本バージョンは、OAuth認証には対応していません。アプリパスワード認証のみ対応しています。
- Blueskyの仕様によりEmbed(埋め込み)は1つに制限されているため、画像とリンクカードを同時に投稿することはできません。

## [v2.0.0] - 2025-12-18

### v2 メジャーアップデート

#### 概要
- StreamNotify v2 をリリースしました。
- アーキテクチャを刷新し、プラグインシステム、GUI、データベース管理を導入しました。

#### 主な機能

**コア機能**
- **プラグインアーキテクチャ**: 機能拡張を容易にするプラグインシステムを導入。
- **SQLite データベース**: 動画情報や投稿状態を `data/video_list.db` で管理。
- **Asset マネージャー**: テンプレートや画像を `Asset/` ディレクトリから自動配置。
- **GUI 統合**: Tkinter ベースの管理画面で動画選択、投稿、削除、設定確認が可能。

**動作モード**
- **通常モード**: RSS取得と手動投稿。
- **自動投稿モード**: RSS取得と自動投稿（`BLUESKY_POST_ENABLED=true`）。
- **ドライランモード**: 投稿をシミュレート。
- **収集モード**: RSS取得のみ。

**連携機能**
- **YouTube RSS**: チャンネルの RSS フィードを定期監視。
- **サムネイル自動取得**: 新着動画のサムネイルを自動ダウンロード。
- **Bluesky 投稿**: 動画情報の自動投稿（Rich Text Facet 対応）。

#### ファイル構成
- `main_v2.py`: エントリポイント
- `v2/`: v2 ソースコード一式
- `v2/Asset/`: アセットリソース
- `v2/plugins/`: プラグインディレクトリ

## [v1.0.0] - 2025-12-15

### 初期リリース

#### 概要
- StreamNotify の最初のバージョンをリリースしました。
- YouTube チャンネルの新着動画を Bluesky に自動投稿する基本機能を提供。

#### 主な機能

**YouTube RSS フィード取得**
- 指定した YouTube チャンネルの新着動画を定期的に取得。
- 動画タイトル、URL、公開日時を取得可能。

**Bluesky 投稿機能**
- YouTube の新着動画情報を Bluesky に投稿。
- 投稿内容は固定フォーマットで構成。

**ログ機能**
- 投稿履歴をローカルに記録。
- エラー発生時のデバッグ情報を出力。

#### 注意事項
- 本バージョンは基本機能のみを提供しており、拡張性は限定的です。
- プラグインアーキテクチャは次期バージョンで導入予定です。
